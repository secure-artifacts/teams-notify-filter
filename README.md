# Teams 通知过滤器

## 功能
- **私人消息始终通知**
- **只有「通知文件夹」里的群组才会提醒**
- 其他群组默认静默
- 在 Teams 页面右下角 **「通知管理」** 面板里直接操作，无需手打群名

## 使用
1. `chrome://extensions/` 重新加载插件
2. 打开 `https://teams.microsoft.com/` 并登录
3. 点击右下角 **「通知管理」**
4. 在 **「其他群组」** 里点击 **「移入通知」**，或拖拽到 **「通知文件夹」**
5. 私人消息在顶部单独显示，始终会通知

## 说明
- 默认所有群组都在「不通知」区，只有你移入通知文件夹的才会提醒
- 若仍收到 Teams 自带通知，请在 Teams 设置 → 通知 里关闭浏览器通知
